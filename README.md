# eBPF PoC Template

Generic template for eBPF program development using libbpf and bpftool.

## Requirements

- clang / llvm
- gcc
- libbpf-dev
- linux-headers
- bpftool
- bear *(optional, for compile_commands.json)*

**Debian / Ubuntu**
```bash
sudo apt install clang llvm gcc libbpf-dev linux-headers-$(uname -r) linux-tools-$(uname -r)
```

**Fedora / RHEL / CentOS**
```bash
sudo dnf install clang llvm gcc libbpf-devel kernel-headers kernel-devel bpftool
```

**Arch Linux**
```bash
sudo pacman -S clang llvm gcc libbpf linux-headers bpf
```

**openSUSE**
```bash
sudo zypper install clang llvm gcc libbpf-devel kernel-devel bpftool
```

## Project Structure

```
.
├── src/
│   ├── <target>.bpf.c      # eBPF program (kernel space)
│   ├── <target>.c          # Userspace program
│   └── <target>.h          # Shared headers (optional)
├── include/                # Additional headers
├── build/                  # Build artifacts (generated)
│   ├── vmlinux.h           # BTF auto-generated
│   ├── <target>.bpf.o      # Compiled eBPF object
│   ├── <target>.skel.h     # Skeleton generated by bpftool
│   └── <target>            # Final binary
├── libs/
│   ├── libbpf/             # Submodule
│   └── bpftool/            # Submodule
├── Makefile
├── .gitmodules
└── README.md
```

## Usage

### Default build

```bash
make
```

Compiles the target defined by `TARGET` (default: `network-trace`).

### Build with a custom target

```bash
make TARGET=my-poc
```

Expected source files are `src/my-poc.bpf.c` and `src/my-poc.c`.

### Available targets

| Command | Description |
|---|---|
| `make` | Full build (default) |
| `make TARGET=<n>` | Build with custom target |
| `make clean` | Remove build artifacts |
| `make install` | Install binary to `/usr/local/bin` |
| `make compile_commands` | Generate `compile_commands.json` for IDE |
| `make help` | Show help |

### Examples

```bash
# Build and install
make TARGET=edr-agent
make TARGET=edr-agent install

# Regenerate compile_commands.json (requires bear)
make TARGET=edr-agent compile_commands
```

## Starting a new PoC

```bash
# Clone the template with submodules
git clone --recurse-submodules https://github.com/<org>/ebpf-poc-template my-poc
cd my-poc

# Rename source files
mv src/network-trace.bpf.c src/my-poc.bpf.c
mv src/network-trace.c     src/my-poc.c

# Build
make TARGET=my-poc
```

## Submodules

```bash
# Initialize after cloning without --recurse-submodules
git submodule update --init --recursive

# Update submodules
git submodule update --remote
```
